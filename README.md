# Kindergarten Management System

A web-based system designed to help kindergartens manage students, teachers, meals, and tuition fees.

The platform improves school administration by organizing important information in a centralized system.

## Features

- Manage student information
- Manage teacher information
- Meal planning and nutrition management
- Tuition fee management
- Administrative management for schools
